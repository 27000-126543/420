## 1. 产品概述

城市数字孪生交互可视化平台（CityTwin）——面向超大规模城市的实时3D数字孪生系统，汇聚IoT传感器、摄像头、社交媒体及气象台多源动态数据，在城市级/区域级/建筑内部级多尺度3D场景中实现智能交通调度、环境监测预警、能耗异常检测，支撑市-区-街道-企业四级权限管控与全流程事件处置协同。

- 目标用户：城市管理者、应急指挥中心、环保/交通/能源部门、区级/街道管理员、入驻企业
- 核心价值：将分散的城市数据统一到3D孪生空间，实现"一屏观全城、一网管全域"的智慧城市治理范式

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 市级管理员 | 系统分配 | 全城数据图层、全流程事件审批、全局调度 |
| 区级管理员 | 上级分配 | 本区数据图层、本区事件审批与处置 |

### 2.2 功能模块

1. **3D城市孪生主场景**：多尺度3D城市场景、自由漫游缩放、图层切换、实时数据叠加
2. **智能交通调度**：实时路况热力图、路径推荐、红绿灯动态调度模拟
3. **环境监测预警**：空气/噪声/水质热力图、预警阈值触发、应急响应流程
4. **能耗分析**：建筑群能耗实时监测、分钟级趋势图、异常点自动标记
5. **事件处置中心**：事件检测-上报-派单-处置-反馈全流程、多人协同标注
6. **权限管控面板**：四级角色管理、数据图层授权、操作审计日志

## 3. 核心流程

### 3.1 事件处置全流程

用户在3D场景中发现异常或系统自动检测到预警后，进入事件处置流程：自动检测异常 → 系统生成预警事件 → 街道管理员审核上报 → 区级管理员审批派单 → 处置人员执行并反馈 → 事件闭环归档。

```mermaid
flowchart TD
    "自动检测异常" --> "系统生成预警事件"
    "系统生成预警事件" --> "街道管理员审核上报"
    "街道管理员审核上报" --> "区级管理员审批派单"
    "区级管理员审批派单" --> "处置人员执行反馈"
    "处置人员执行反馈" --> "事件闭环归档"
    "区级管理员审批派单" -->|"退回"| "街道管理员审核上报"
```

### 3.2 智能交通调度流程

实时采集交通数据 → 计算路段流量指数 → 生成热力图与路径推荐 → 调度算法生成红绿灯配时方案 → 下发执行 → 效果评估反馈。

```mermaid
flowchart TD
    "实时采集交通数据" --> "计算路段流量指数"
    "计算路段流量指数" --> "生成热力图与路径推荐"
    "计算路段流量指数" --> "调度算法生成配时方案"
    "调度算法生成配时方案" --> "下发执行"
    "下发执行" --> "效果评估反馈"
    "效果评估反馈" --> "实时采集交通数据"
```

## 4. 用户界面设计

### 4.1 设计风格

- **整体风格**：赛博朋克工业风，深邃暗色基调搭配霓虹高亮，营造科技感十足的指挥中心氛围
- **主色调**：深空黑（#0a0e1a）为底色，电光蓝（#00f0ff）为主强调色，琥珀橙（#ff6b2b）为预警/告警色，翡翠绿（#00ff88）为正常/安全色
- **按钮风格**：微透明磨砂玻璃质感，hover时边框发光，圆角4px
- **字体**：数据展示用等宽字体 JetBrains Mono / Source Code Pro，UI文案用思源黑体 Noto Sans SC
- **布局风格**：全屏3D场景为主，左/右/底部悬浮半透明面板，顶部极简导航栏
- **图标风格**：线性描边图标，2px线宽，霓虹发光效果
- **动画效果**：数据流粒子动画、热力图渐变脉动、预警闪烁、面板滑入滑出

### 4.3 响应式设计

- 桌面优先设计，主要面向大屏/指挥中心场景（1920×1080及以上）
- 中等屏幕（1366×768）保持核心功能，面板可折叠
- 不支持移动端（指挥场景需大屏操作）

### 4.4 3D场景指导

- **环境/氛围**：深蓝色天空盒，底部雾效营造城市纵深感，夜景模式为主
- **光照**：冷色调环境光 + 暖色点光源模拟城市灯光，建筑窗户自发光
- **摄像机**：OrbitControls自由旋转缩放，最小距离50（建筑级），最大距离5000（城市级），默认倾斜45度俯瞰
- **构图与焦点**：城市中心为默认焦点，标志性建筑/异常区域脉冲光环引导注意力
- **交互与动画**：点击建筑聚焦并弹出信息卡，双击进入建筑内部视图，滚轮缩放切换LOD
- **后处理**：Bloom辉光效果（霓虹灯光）、景深（远景模糊）、色调映射
- **性能预算**：目标60fps，建筑实例化渲染，热力图使用纹理而非几何体，LOD距离切换
